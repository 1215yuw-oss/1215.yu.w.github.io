# MotorSync 基礎保養知識庫 (Maintenance Rules)

本文件為 MotorSync 智能推算系統的核心業務邏輯基礎 (基於常見 PTT 機車版與原廠通用建議綜合彙整)，用於定義各組件之生命週期及警示閥值。

## 👉 機車定期保養一覽表（通用版）

保養機車應依「里程數」與「時間」雙重標準進行。以下為核心計算常數：

| 保養項目 | 建議公里數 (km) | 保養目的 |
| --- | --- | --- |
| **機油** | 1,000 | 潤滑引擎、降溫（最重要） |
| **齒輪油** | 2,000 - 3,000 | 潤滑齒輪箱、減少異音 |
| **空氣濾清器** | 3,000 - 5,000 | 確保引擎呼吸潔淨空氣，提升動力 |
| **傳動組清潔** | 5,000 - 10,000 | 清除皮帶粉塵、優化加速 |
| **煞車皮/來令片** | 5,000 - 10,000 (視情況) | 檢查厚度，低於限度即更換 |
| **火星塞** | 10,000 | 確保點火順暢，防止引擎發動不順 |
| **汽油濾芯** | 10,000 - 15,000 | 過濾汽油雜質 |
| **煞車油** | 10,000 - 20,000 | 確保煞車系統靈敏 |
| **電瓶** | 10,000 - 15,000 (或 1-2 年) | 檢查電壓，確保啟動正常 |
| **傳動皮帶** | 15,000 - 20,000 | 檢查龜裂、預防斷裂 |
| **輪胎** | 依胎紋深度 (小於0.8mm須換) | 確保行車抓地力與安全 |

## 🚨 重點注意事項 (邊界條件)

在實踐系統運算時，必須考慮以下情境特例：

1. **基礎巡檢 (每 1,000 km)**
   - 每次更換機油時，應順便建立檢查：胎壓、煞車、燈光 的微紀錄。
2. **新車第一次保養 (300 km 特例法則)**
   - 新車引擎磨合期會產生較多金屬鐵屑，因此機油與齒輪油通常在落地 **300km** 時必須做第一次強制更換。
3. **低使用率車輛 (長期未騎)**
   - 若長時間未使用，建議保養項目同樣按「時間月份」計算，不可死守里程數，並建議定期牽去車行清潔噴油嘴。
4. **極端環境用車 (高風險項目)**
   - 若行駛砂石路面比例高、或是長期處於高粉塵工業區，應縮短空氣濾清器更換週期 (由 5,000km 降至 3,000km 以下)。 

*(本文件作為 MotorSync 架構演算法之底層基準)*
